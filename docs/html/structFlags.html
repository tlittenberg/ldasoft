Detailed Description

Run flags set or changed from default values when parsing command line for gb_mcmc and gb_catalog.

Descriptions of each flag includes default settings and command line arguments to change/set flags. Boolean flags are defined as integers with 0==FALSE and 1==TRUE.

To see how the defaults are set and then adjusted according to the command line, see parse().
