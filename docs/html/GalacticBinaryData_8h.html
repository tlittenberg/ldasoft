Removes known sources which are outside of analysis window but whose signal power may extend into the region of interest.

Signal parameters are read from file picked input using the --catalog command line option. Sources with \(f_0<f_{\rm min}\) but \(f_0 + (B/2)/T > f_{\rm min}\) are removed from the data.

Any sources in the catalog with \( f_{\rm min}\leq f_0 \leq f_{\rm max} \) (including window padding) are not used in the subtraction.

Injection routine for EM-known binaries.

EM observations do not provide information about polarization angle \(\psi\) or (currently) initial phase \(\varphi_0\). Those two parameters are missing from the injection files and drawn from their priors.
